1 Basics

  • MTF transform [1] performs a change in the representation of the symbols of a sequence, where those symbols that have a high probability of occurrency are “moved” in the source alphabet towards decreasing positions. Therefore, MTFT inputs N symbols and output N symbols.
  • The output sequence has a probability density function which follows an exponential distribution with a slope λ:
    f(x) = λe−λx if x ≥ 0; 0 if x < 0. (1)

    where λ depends on the probability of ocurrence of the input symbols.

2 Forward transform

  1. Create a list L with the symbols of the source alphabet, where
    L[s] ← s;0 ≤ s ≤ r.

  2. While the input is not exhausted:
    1. s ← next input symbol.
    2. c ← position of s in L (L[c] = s).
    3. Output ← c.
    4. Move s to the front of L.

    2. s ← L[c].
    3. Output s.
